# Principes de Design - Charlie Osborne

## Caractéristiques Visuelles Clés

### Palette de Couleurs
- **Fond** : Noir profond (#000000 ou très proche)
- **Accent principal** : Dégradé rouge/orange vibrant (du bas vers le haut)
- **Texte** : Blanc pur pour contraste maximal
- **Accents secondaires** : Rose/magenta, vert, bleu, jaune (codes couleur sur les boutons de navigation)

### Typographie
- **Titre principal** : Très grande, sans-serif, gras, blanc
- **Sous-titres** : Mélange de texte normal et italique pour créer du contraste
- **Taille** : Hiérarchie très prononcée (hero très grand)
- **Espacement** : Généreux, minimaliste

### Mise en Page
- **Grille** : Colonnes visibles/invisibles, très structurée
- **Navigation** : Fixe en haut, minimaliste, avec codes couleur pour chaque section
- **Hero Section** : Pleine hauteur avec dégradé spectaculaire
- **Sections** : Alternance de sections claires et avec dégradés

### Animations & Interactions
- **Hover effects** : Changements subtils de couleur/opacité
- **Transitions** : Fluides, pas de saccades
- **Scroll** : Animations au scroll (parallax léger, révélation progressive)
- **Micro-interactions** : Boutons réactifs avec feedback visuel

### Éléments de Design
- **Lignes de grille** : Visibles, créent une structure géométrique
- **Espacements** : Très généreux, respire beaucoup
- **Ombres** : Minimales ou absentes
- **Bordures** : Fines, précises
- **Icônes/Badges** : Codes couleur pour catégoriser les sections

## Structure de Navigation

1. **Header fixe** : Logo + Navigation horizontale + Codes couleur
2. **Sections principales** : WORK, SERVICES, ABOUT, TESTIMONIALS, CONNECT
3. **CTA** : "LET'S CHAT" répété (appel à l'action fort)

## Stratégie pour Adaptation (Alaeddine Dehimi)

### Changements pour Éviter Copie
1. **Palette de couleurs** : Remplacer rouge/orange par **bleu/teal/gris** (plus tech/professionnel)
2. **Dégradé hero** : Adapter le dégradé avec les nouvelles couleurs
3. **Typographie** : Garder sans-serif mais changer la police (ex: Inter → Sora ou Outfit)
4. **Contenu** : Adapter les sections (WORK → PROJETS, SERVICES → EXPERTISE, ABOUT → À PROPOS)
5. **Codes couleur** : Créer une nouvelle palette pour les badges de navigation

### Éléments à Conserver
- Structure minimaliste et épurée
- Hiérarchie typographique forte
- Animations fluides et subtiles
- Grille visible/structurée
- Espacement généreux
- Navigation fixe avec codes couleur
- CTA fort et visible

### Sections à Adapter
- **WORK** → **PROJETS** (PLM, ERP, Industrie 4.0)
- **SERVICES** → **EXPERTISE** (6 domaines)
- **ABOUT** → **À PROPOS** (Bio + valeurs)
- **TESTIMONIALS** → **LABS** (Idées innovantes avec accès)
- **CONNECT** → **CONTACT** (Calendly + Réseaux sociaux)
